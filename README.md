# Backend

## Requisitos previos

- Node.js (v20 o superior)
- pnpm o npm

## Instalación

```bash
pnpm install
```

## Ejecutar la aplicación

### Modo desarrollo

```bash
pnpm dev
```

## Variables de entorno

Crea un archivo `.env` en la raíz del proyecto con las siguientes variables:

```env
PORT=3000
DATABASE_URL=url_de_base_de_datos
```
